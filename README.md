# Electron Player

A Smart music player developed with ElectronJS & ReactJS
==================================

HOW TO RUN
========
1. Clone the project from git.
2. Go to 'electron-player' directory.
3. Run `npm install` or `yarn`.
4. Run `npm run build`.
5. Run `npm start`.
